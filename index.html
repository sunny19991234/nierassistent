API sleutel instellen

De NierAssistent werkt via de Claude API van Anthropic. Je hebt een persoonlijke API sleutel nodig om te starten.

Maak er gratis één aan op console.anthropic.com. Je betaalt alleen per gebruik (~€0,01 per gesprek).

🔒 Je sleutel wordt alleen lokaal op jouw telefoon opgeslagen (localStorage). Hij wordt nooit verzonden naar een server van ons — alleen rechtstreeks naar Anthropic.
🫘

Waar kan ik je mee helpen?

Stel me alles over nieren, niertransplantatie, herstel, medicatie of voeding.

⚕️ Informatief — geen medisch advies. Raadpleeg altijd je behandelend arts.
