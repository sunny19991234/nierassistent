Sunny's NierRobot

API sleutel instellen

De NierRobot werkt via de Claude API van Anthropic. Maak een gratis sleutel aan op console.anthropic.com. Je betaalt alleen per gebruik.

Je sleutel wordt alleen lokaal op jouw telefoon opgeslagen. Hij wordt nooit verzonden naar een tussenliggende server — alleen rechtstreeks naar Anthropic.
